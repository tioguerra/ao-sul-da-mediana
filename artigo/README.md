# Materiais do artigo

Este diretório contém o banco de evidências, referências, dados e figuras do *position paper*.

- [`banco-de-evidencias.md`](banco-de-evidencias.md): levantamento dirigido, matriz de evidências, contraprovas e recomendações editoriais.
- [`referencias-chave.bib`](referencias-chave.bib): referências em BibTeX.
- [`dados-figuras.csv`](dados-figuras.csv): valores transcritos de estudos publicados.
- [`gerar-graficos.py`](gerar-graficos.py): reprodução das quatro figuras bibliográficas.
- [`mapa-cultural/`](mapa-cultural/): auditoria própria, mapas, respostas e análises de distância.

Execute `make figures` na raiz para regenerar as figuras deste diretório.
